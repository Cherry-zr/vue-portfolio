# 视觉标记

视觉依据为用户授权的视频、项目报告中的移动端截图和现有页面。

- 目标视口：390 CSS px 优先；最大内容宽度 430 px。
- 字体：系统中文无衬线字体，正文 14-16 px，标题 17-26 px。
- 主色：`#ff4d58`；发布按钮使用橙红渐变。
- 正文：`#222222`；次要文字：`#8f8f8f`；弱分割线：`#f1f1f1`。
- 页面背景：主要内容白色；信息卡使用 `#f7f7f7` 或低饱和浅色。
- 圆角：搜索框 22 px，内容卡 10-14 px，按钮 18-22 px。
- 头部：白色固定/粘性区域，图标按钮约 40 px。
- 首页频道：文字等分排列，选中项黑色加粗并带红色短下划线。
- 内容流：双列瀑布流；图片比例取自接口，标题最多两行。
- 底部导航：固定白底，中间圆形橙红发布按钮；无接口写操作仅显示禁用说明。
- 动效：页面切换 220-280 ms；尊重 `prefers-reduced-motion`。
